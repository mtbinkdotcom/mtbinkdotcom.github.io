Author:MUHAMMAD TARMIZI BIN KAMARUDDIN / mtbink.com / @mtbinkdotcom

    • 8.1. list

8.1. list¶

  1. jQuery - Various JavaScript utilities.

    Note:

    Download the *min.js but rename to just jquery.js

  2. Underscore.js - Contains great array functions

    Note:

    Download the *min.js but rename to just underscore.js

  3. Material Design Lite - The lite version for Material Design.

    Note:

    1. only these are needed:
      1. material.min.js
      2. material.min.css

  4. Material Icons - Material Icons

    Note:

    1. Download .zip at https://github.com/google/material-design-icons/ and extract it. Only folder iconfont needed.
    2. only these are needed:
      1. MaterialIcons-Regular.eot
      2. MaterialIcons-Regular.woff2
      3. MaterialIcons-Regular.woff
      4. MaterialIcons-Regular.ttf

  5. odometer - Animate counter.

    Note:

    Only “odometer.min.js” and “theme/odometer-theme-minimal.css” are needed

  6. string.js - Various string function.

    Note:

    Only dist/string.min.js needed

  7. jsUri - Fluent javascript uri/url parsing and manipulation.

    Note:

    Only Uri.js needed

  8. jQuery UI datepicker - Date picker.

    Note:

    1. Only “jquery-ui.min.js” and “jquery-ui.min.css” are needed
    2. I only need these in Download Builder:
      • Core
      • Widgets/Datepicker
      • Effects/Effects Core
      • Effects/Drop Effect
      • select No Theme

  9. jQuery qrcode - Generate QR code.

    Note:

    Only jquery.qrcode.min.js needed

  10. MathJax - Show math document in browser.

    Note:

    1. Rename top folder to “MathJax”.
    2. We need to cut down this monster folder of 31791 files (53.3 MB):
      • delete folder docs
      • delete folder images
      • delete folder localization
      • delete folder test
      • delete folder unpacked
      • delete folder extensionsMathML
      • in the folder fontsHTML-CSS, just leave TeX
      • in the folder fontsHTML-CSSTeX, just leave woff
      • in the folder jaxinput, just leave TeX
      • in the folder jaxoutput, just leave HTML-CSS
      • in the folder jaxoutputHTML-CSSfonts, just leave TeX
      • in the top folder, only MathJax.js is important, delete other files *.* (not other folders)
    3. Now we have just 198 files (3.94 MB) as of 2013-11-11T05:39:02-08:00

  11. Table Sorter - Sort table

    Note:

    Only tablesort-min.js needed

  12. jsdiff - A javascript text differencing implementation.

    Note:

    View-source, only diff.js needed

  13. Roboto font - Roboto font

    Note:

    All TTF fonts are needed.

  14. moment.js - A JavaScript date library for parsing, validating, manipulating, and formatting dates.

    Note:

    Only moment.min.js needed
