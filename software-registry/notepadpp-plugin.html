5. Notepad++ plugin¶

Author:MUHAMMAD TARMIZI BIN KAMARUDDIN (002321867-D) / mtbink.com / @mtbinkdotcom

5.1. Inside Plugin Manager¶

  1. Plugins > Plugin Manager > Show Plugin Manager > Available

    1. Code alignment - To align code at certain character, for example, equal symbol (=).

    2. FingerText - Text expander.

    3. JSTool - JavaScript code formatter

      Note:

      JSTool > Options... > Use 4 spaces instead of tab (t).

    4. XML Tools - Formatting XML-based files.

5.2. Outside Plugin Manager¶

  1. NppJavaScript / JavaScript Macro - JavaScript macro programming for Notepad++

    Note:

    Copy NppJavaScript.dll to the plugins directory and restart Notepad++.
