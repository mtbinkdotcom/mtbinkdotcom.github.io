      5. Adjust Volume Control as long as possible.

      6. At the bottom, View > by artist.

      7. Result:

    3. File > Preferences > Media Library > Add... and locate MP3 folder.

    4. Download: http://www.foobar2000.org/components/view/foo_dsp_effect then drag that file to the window: File > Preferences > Components. Click Apply.
