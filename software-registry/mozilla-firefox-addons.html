4. Mozilla Firefox Add-ons¶

Author:MUHAMMAD TARMIZI BIN KAMARUDDIN / mtbink.com / @mtbinkdotcom
  • 4. Mozilla Firefox Add-ons
    • 4.1. list

4.1. list¶

  1. Auto Refresh - Auto refresh web page.

  2. Check4Change - Monitor text changes in a webpage.

  3. Cleanest Addon Manager - Compact the Addon Manager.

  4. ColorZilla - Check the colour of an element in a web page.

  5. Context Search - Right-click search menu.

  6. CSS Usage - Firebug extension to view which CSS rules are actually used.

  7. DownThemAll! - Download accelerator and manager.

  8. FEBE - Backuo extensions, profiles, sessions.

  9. Firebug - Web page development tools.

  10. Firefinder for Firebug - Finds HTML elements matching chosen CSS selector(s) or XPath expression.

  11. FireQuery - Firebug plugin for jQuery development.

  12. Flagfox - Shows flag icon in the address bar.

  13. Google search link fix - Transform all the links in Google search page into direct links (no more very long gibberish links).

  14. Greasemonkey - Customize any web page using JavaScript.

  15. iMacros for Firefox - Firefox automation.

  16. Linkification - Converts text links into genuine, clickable links.

    Note:

    1. Options > Functionality > tick Open Link in New Tab
    2. Options > Functionality > tick Open New Tab in Background

  17. MeasureIt - Check the width of an element in a web page.

  18. Pearl Crescent Page Saver Basic - Web page screenshot.

  19. QuickJava - Enable or disable JavaScript / Java / Flash / Silverlight / cookies / images / animated images / CSS

  20. ReloadAll! - Reload all/some tabs.

  21. Restartless Restart - Bring restart button.

  22. Save Session - Save session and exit

  23. Scrollbar Search Highlighter - An add-on which highlights search terms in the scrollbar(s).

  24. SearchPreview - Provides thumbnail in search result page.

  25. SEO Status PageRank/Alexa Toolbar - Alexa ranking.

  26. Session Manager - Session manager.

  27. Status-4-Evar - Bring back the status bar.

  28. Tab Counter - Show tab counter.

  29. tabs2list - List all the tab(s).

  30. User Agent Switcher - User agent switcher.

  31. Video DownloadHelper - Download YouTube/Flash video.

  32. View Source Chart - Inspect the DOM graphically.
