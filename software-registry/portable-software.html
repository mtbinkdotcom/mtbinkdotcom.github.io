6.1. free software¶

  1. addon-sdk - Create Mozilla Firefox Add-ons through python.

  2. AudacityPortable - Audio file editor, apply sound effect, truncate, join, reverse, convert FLV to MP3 (by using LAME MP3/FFmpeg), extract audio from video (using FFmpeg).

    Note:

    1. Setup these: * Edit > Preferences... > Libraries > MP3 Library > Locate... then find lame_enc.dll * Edit > Preferences... > Libraries > FFmpeg Library > Locate... then find avformat-52.dll
    2. Locate those files in AudacityPortable\App\LAME and AudacityPortable\App\FFMpeg respectively.
    3. Sometimes you have to redo the setup for those 2 files.

  3. AutoHotkey - Keyboard and mouse automation.

    Note:

    1. Click Express Installation
    2. After finished installing, the folder C:\Program Files\AutoHotkey is now portable.
    3. Do not uninstall the C:\Program Files\AutoHotkey, it will makes file input output very slow (happens on my office laptop).

  4. CamStudioPortable - Screen recorder.

  5. cdrtfePortable - CD/DVD burner.

  6. CuteMarkEd - Markdown editor, convert to HTML or PDF.

    Note:

    Rename top folder to cutemarked

  7. DiffpdfPortable - Compare 2 PDF files.

  8. FFmpeg - Extract audio from video (using Audacity Portable), extract an image from a video via Command Prompt.

    Note:

    1. In the section FFmpeg Binary compatible with Audacity 1.3.13, 2.x or later on Windows (THIS IS NOT LAME!):, download the ZIP version - here. Usually the download is http://lame3.buanzo.com.ar/FFmpeg_v??_for_Audacity_on_Windows.zip.
    2. Rename top folder to FFmpeg
    3. It is advisable to mirror the content into AudacityPortable\App\FFMpeg

  9. FileZilla Portable - FTP client.

    Note:

    1. When extracting, no need the Additional Languages
    2. Resetup FTP login.

  10. GIMPPortable - Image editor.

    Note:

    When extracting, no need the Additional Languages

  11. graphviz - Graph visualization.

  12. ImageMagick - Manipulate image through command line.

  13. InkscapePortable - Create vector image.

  14. KeePass - Password manager.

    Note:

    1. Open the latest version folder.
    2. Download KeePass-??.zip
    3. Rename top folder to KeePass
    4. Need to relocate database file.

  15. lame - Convert FLV to MP3 (using Audacity Portable).

    Note:

    1. In the section ZIP OPTION:, download the libmp3lame-win-??.zip.
    2. Rename top folder to lame
    3. It is advisable to mirror the content into AudacityPortable\App\LAME

  16. LICEcap - Screen recorder directly output animated GIF.

  17. Mozilla Rhino - JavaScript through Command Prompt.

    Note:

    1. Rename top folder to Mozilla Rhino.
    2. Only js.jar is important; all others can be deleted.

  18. Pandoc - Multi-output markdown-based converter.

    Note:

    1. Install the pandoc-??.msi
    2. After finished installing, the folder C:\Users\??\AppData\Local\Pandoc is now portable
    3. Uninstall the C:\Users\??\AppData\Local\Pandoc

  19. PeaZipPortable - File compressor and de-compressor.

  20. phantomjs - Scriptable headless WebKit using Command Prompt.

    Note:

    Rename top folder to phantomjs

  21. php - PHP through Command Prompt.

    Note:

    1. Download the VC11 x86 Thread Safe ZIP file.
    2. Rename top folder to php
    3. php\php.ini * change date.timezone = Asia/Kuala_Lumpur * make sure extension_dir = "ext/" * make sure extension=php_openssl.dll
    4. Need to install msvcr110.dll. Choose VSU4\vcredist_x86.exe.

  22. Sekaiju - Beautiful MIDI sequencer software.

  23. VLCPortable - Video player.

    Note:

    When extracting, no need the Additional Languages

  24. WinDjViewPortable - DJVU file reader.

  25. WinMergePortable - Compare 2 text files.

  26. wkhtmltopdf - WebKit HTML-to-PDF using Command Prompt.

    Note:

    1. Open the latest version folder.
    2. Download wkhtmltox-win32_??-03c001d.exe
    3. After finished installing, the folder C:\Program Files (x86)\wkhtmltopdf is now portable
    4. Uninstall the C:\Program Files (x86)\wkhtmltopdf

  27. FastCopy - File copier (very fast).

    Note:

    Rename top folder to FastCopy

  28. FirefoxPortable36 - Mozilla Firefox portable 36.

  29. FirefoxPortable38 - Mozilla Firefox portable 38.

  30. Greenshot - Taking screenshot, annotate image, upload to imgur.

    Note:

    1. Open the latest version folder.
    2. Download Greenshot-NO-INSTALLER-??.zip
    3. Rename top folder to Greenshot
    4. Settings > Capture, ensure 0 Milliseconds to wait before capture
    5. Settings > Output, ensure JPEG quality is 100

  31. Notepad++ - Text editor.

    Note:

    Notepad++ special setup

6.2. freeware¶

  1. nirsoft/cports-x64 - Monitoring Opened TCP/IP network ports / connections.

  2. nirsoft/netbscanner - NetBScanner is a network scanner tool that scans all computers in the IP addresses range you choose, using NetBIOS protocol.

  3. nirsoft/networkconnectlog - Log connected/disconnect devices on your network.

  4. nirsoft/networktrafficview-x64 - Monitor the traffic on your network adapter.

  5. nirsoft/nircmd-x64 - Alert, beep, change brightness, change volume, trayballoon, all using command line.

  6. nirsoft/smsniff-x64 - Capture TCP/IP packets on your network adapter.

  7. nirsoft/tabletextcompare - Compare 2 text files.

  8. nirsoft/whatishang-x64 - What software is currently hang.

  9. nirsoft/wirelesskeyview - Recover lost WEP/WPA key/password stored by Wireless Zero Configuration service.

  10. nirsoft/wirelessnetconsole - View wireless networks information.

  11. nirsoft/wnetwatcher - Show who is connected to your wireless network.

  12. TreeSizeFreePortable - Scan directory sizes and analyze usage.

  13. SpaceSniffer - Scan directory and show size graphically.

  14. WizTree - Scan directory and list down all the files by size.

  15. systernals/Autoruns - Manage autoruns.

    Note:

    Rename folder Autoruns to autoruns

  16. systernals/BGInfo - Show computer info on the wallpaper.

  17. systernals/ProcessExplorer - Process explorer.

  18. BalabolkaPortable - Text-to-speech converter.

  19. cpu-z_1.74-en - CPU info.

  20. Everything - Search filename in an NTFS drive.

    Note:

    1. Rename top folder to Everything
    2. Rename executables to Everything.exe

  21. FoxitReaderPortable - PDF reader.

  22. HJSplit - File splitter/joiner.

  23. IcoFXPortable - Create icon file.

  24. IrfanViewPortable - Animated GIF viewer.

    Note:

    When extracting, no need the Additional Languages

  25. MAMPlayer - Beautiful MIDI sequencer software.

  26. notifu - Notify using trayballoon.

  27. Portable Java - Java environment.

    Note:

    1. Rename the Java folder to Portable Java and this is now the top folder.

  28. ShowKeyPlus - Windows product key finder and validation checker.

  29. VisualBoyAdvance - VisualBoyAdvance

  30. XMediaRecode - Convert video and audio.

    Note:

    1. Rename top folder to XMediaRecode
    2. Rename executables to XMediaRecode.exe

  31. Coral/Uniform Server - PHP web server.

    Note:

    Coral/Uniform Server special setup

  32. MiKTeX Portable - TeX environment.

  33. WinPython-64bit-2.7.10.3 - Python 2 environment.

  34. WinPython-64bit-3.4.3.5 - Python 3 environment.
