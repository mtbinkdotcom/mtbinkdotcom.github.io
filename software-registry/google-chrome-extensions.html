2.1. list¶

  1. Alexa Traffic Rank - Alexa Traffic Rank.
  2. Auto Refresh Plus - Auto refresh web page.
  3. Checker Plus for Gmail - Show the total unread emails in Gmail inbox and notify the user for new incoming email.
  4. Checker Plus for Google Calendar - Google Calendar checker.
  5. Google Cast (Beta) - Google Cast.
  6. Markdown Here - Use Markdown when composing email in Gmail.
  7. RSS Subscription Extension (by Google) - Provides Google Chrome to read RSS file.
  8. SearchPreview - Provides thumbnail in search result page.
  9. Session Buddy - Session import/export.
