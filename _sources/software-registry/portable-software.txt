Portable software
=================

:Author: |author|



free software
-------------

#.  `addon-sdk <https://developer.mozilla.org/en-US/Add-ons/SDK/>`_ - Create Mozilla Firefox Add-ons through python.

#.  `AudacityPortable <http://portableapps.com/apps/music_video/audacity_portable>`_ - Audio file editor, apply sound effect, truncate, join, reverse, convert FLV to MP3 (by using LAME MP3/FFmpeg), extract audio from video (using FFmpeg).

    **Note**: 

    #.  Setup these:  
        * **Edit > Preferences... > Libraries > MP3 Library > Locate...** then find **lame_enc.dll**  
        * **Edit > Preferences... > Libraries > FFmpeg Library > Locate...** then find **avformat-52.dll**
    #.  Locate those files in **AudacityPortable\\App\\LAME** and **AudacityPortable\\App\\FFMpeg** respectively.
    #.  Sometimes you have to redo the setup for those 2 files.

#.  `AutoHotkey <http://ahkscript.org>`_ - Keyboard and mouse automation.

    **Note**: 

    #.  Click **Express Installation**
    #.  After finished installing, the folder **C:\\Program Files\\AutoHotkey** is now portable.
    
    #.  **Do not uninstall** the **C:\\Program Files\\AutoHotkey**, it will makes file input output very slow (happens on my office laptop).

#.  `CamStudioPortable <http://portableapps.com/apps/utilities/camstudio_portable>`_ - Screen recorder.

#.  `CuteMarkEd <https://github.com/cloose/CuteMarkEd/releases>`_ - Markdown editor, convert to HTML or PDF.

    **Note**: 

    Rename top folder to **cutemarked**

#.  `DiffpdfPortable <http://portableapps.com/apps/utilities/diffpdf_portable>`_ - Compare 2 PDF files.

#.  `FFmpeg <http://lame.buanzo.org/>`_ - Extract audio from video (using Audacity Portable), extract an image from a video via Command Prompt.

    **Note**: 

    #.  In the section **FFmpeg Binary compatible with Audacity 1.3.13, 2.x or later on Windows (THIS IS NOT LAME!):**, download the **ZIP version - here**. Usually the download is ``http://lame3.buanzo.com.ar/FFmpeg_v??_for_Audacity_on_Windows.zip``.
    #.  Rename top folder to **FFmpeg**
    #.  It is advisable to mirror the content into **AudacityPortable\\App\\FFMpeg**

#.  `php <http://windows.php.net/download/>`_ - PHP through Command Prompt.

    **Note**: 

    #.  Download the **VC11 x86 Thread Safe** ZIP file.
    #.  Rename top folder to **php**
    #.  **php\\php.ini**  
        * change ``date.timezone = Asia/Kuala_Lumpur``
        * make sure ``extension_dir = "ext/"``
        * make sure ``extension=php_openssl.dll``
    #.  Need to install `msvcr110.dll <http://www.microsoft.com/en-us/download/details.aspx?id=30679>`_. Choose **VSU4\\vcredist_x86.exe**.
