6. Notepad++¶

Author:MUHAMMAD TARMIZI BIN KAMARUDDIN (002321867-D) / mtbink.com / @mtbinkdotcom

6.1. Special setup¶

  1. Edit (menu bar) and tick the following:

    1. > Clipboard history
    2. > Character panel

  2. View (menu bar) and tick the following:

    1. > Show symbol > Show White Space and TAB
    2. > Word wrap
    3. > Document Map
    4. > Function List

  3. Language (menu bar):

    1. Additional syntax highlighting:

      1. .rst
      2. .twig (suitable for jinja2)

      simply download the .xml file and import it through

      Language > Define your language... > Import...

  4. Settings (menu bar):

    1. > Preferences... and update the following:

      1. > General:

        1. > Document List Panel > and tick Show
        2. > Tab Bar > and tick Hide

      2. > Editing:

        1. > Multi-Editing Settings > and tick Enable (Ctrl+Mouse click/selection)
        2. > Vertical Edge Settings:
          1. Tick Show Vertical Edge and choose Line mode
          2. Number of columns set to 80
        3. Tick Enable smooth font

      3. > Default Directory:

        1. > Default Directory (Open/Save) > and choose Remember last used directory
        2. Tick Use new style save dialog (without file extension feature)

      4. > Recent Files History:

        1. > Display > and tick In Submenu

      5. > Language Menu:

        1. > Language Menu > Available items

          I only use these:

          1. Normal Text
          2. PHP
          3. Java
          4. HTML
          5. XML
          6. Batch
          7. INI file
          8. ASP
          9. Visual Basic
          10. JavaScript
          11. CSS
          12. Python
          13. Lua
          14. TeX
          15. Scheme
          16. AutoIt
          17. Inno Setup
          18. YAML
          19. CoffeeScript
          20. JSON
          21. JavaScript (Yes, it is twice appearing)

          All others move it to the Disabled items

      6. > Tab Settings:

        1. > Tab Settings > and tick Replace by space

      7. > Auto-Completion:

        1. > Auto-Insert > and untick everything

      8. > MISC.:

        1. > Clickable Link Settings > and untick Enable
        2. > Highlight Matching Tags> and tick Highlight comment/php/asp zone
        3. Session file ext. set to nses

    2. > Style Configurator... and update the following:

      1. > Global Styles : Global override:
        1. > Font Style > Font Name > and choose Hack
        2. Tick Enable global font

  5. Run (menu bar):

    1. Run Command Prompt against current active file:

      1. Run > Run... > cmd /c "$(FULL_CURRENT_PATH)" > Save...:

      2. CMD > CTRL > F5 > OK:

      3. Now, just press Ctrl + F5 everytime you need to run Command Prompt against current active file.

        Note that the file must be saved first.

  6. Plugins (menu bar):

    1. Uninstall plugin(s):
      1. Plugins > Plugin Manager > Show Plugin Manager > Installed then tick the following:
        1. Converter
        2. NppExport
      2. Click Remove.
    2. Need to reinstall Notepad++ plugin

6.2. Why I do not use these styles ?¶

  1. Default

    White background, already bored looking at VBA

  2. Bespin

    Bracket matching is not prominent at all.

  3. Black board

    Highlighted variable is not prominent.

  4. Choco

    The colouring is almost like Bespin, but the currently selected text range is not prominent at all; here I highlight from 4... to ...Obje actually:

  5. Hello Kitty

    Too pinkish.

  6. HotFudgeSundae

    Highlighted variable is not prominent.

  7. khaki

    Highlighted variable is not prominent.

  8. Mono Industrial

    Whitish font.

  9. Monokai

    The currently selected text range is not prominent at all.

  10. MossyLawn

    Highlighted variable is not prominent.

  11. Obsidian

    The currently selected text range is not prominent at all.

  12. Plastic Code Wrap

    The currently selected text range is not prominent at all.

  13. Ruby Blue

    Highlighted variable is not prominent.

  14. Solarized-light

    Highlighted variable is very dark.

  15. Solarized

    Highlighted variable is not prominent.

  16. Twilight

    The currently selected text range is not prominent at all.

  17. Vibrant Ink

    Highlighted variable is not prominent.

  18. vim Dark Blue

    Too blue.

  19. Zenburn

    Highlighted variable is not prominent.

  20. Currently, Deep Black, Navajo works fine to me.
