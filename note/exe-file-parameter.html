3. EXE file parameter in Command Prompt¶

Author:MUHAMMAD TARMIZI BIN KAMARUDDIN (002321867-D) / mtbink.com / @mtbinkdotcom

3.1. note¶

  1. We use these for all the examples below:
    • folder path (from) = E:\folder1
    • folder path (to) = E:\folder2
    • original file path = E:\file1.xlsx

3.2. explorer.exe¶

3.2.1. Restart explorer¶

@ECHO OFF
TASKKILL /IM explorer.exe /F
START explorer.exe

Do not use /T for explorer.exe.

START explorer.exe is needed to close Command Prompt; explorer.exe is just not enough.

3.2.2. Open folder¶

explorer.exe "E:\folder1"

3.2.3. Open folder and autoselect a file¶

explorer.exe /select,C:\Windows\notepad.exe

3.2.4. Open any file with its default viewer/editor¶

Opening XLSX file with the default Microsoft Excel:

explorer.exe "E:\file1.xlsx"

3.3. Zip a folder¶

3.3.1. using 7z.exe¶

"E:\PeaZipPortable\App\PeaZip\res\7z\7z.exe" a -tzip -mm=Deflate -mmt=on -mx5 "E:\folder1.zip" "E:\folder1"

3.3.2. using PeaZip Portable¶

  1. 1 folder:

    "E:\PeaZipPortable\PeaZipPortable.exe" -add2zip "E:\folder1"

  2. many folders/files:

    "E:\PeaZipPortable\PeaZipPortable.exe" -add2zip "E:\folder1" "E:\folder2" "E:\file1.xlsx"

3.4. Extract ZIP file to a folder (unzip)¶

"E:\PeaZipPortable\PeaZipPortable.exe" -ext2folder  "E:\folder1.zip"

3.5. PHP¶

3.5.1. Start localhost:8080 PHP server¶

"php.exe" -S localhost:8080 -t "E:\folder1"

3.5.2. Passing parameters to PHP script¶

"php-cgi.exe" -f "E:\folder1\p\index.php" p=utility/calculate-age

This is equivalent to opening index.php?p=utility/calculate-age in the browser.

3.6. FastCopy¶

3.6.1. sync 2 folder the most efficient way¶

It will only copies items that have changed date/size.

Any item that only exists in destination folder but not in source folder, will be deleted.

"FastCopy.exe" /no_ui /cmd=sync /bufsize=512 "E:\folder1" /to="E:\folder2"
