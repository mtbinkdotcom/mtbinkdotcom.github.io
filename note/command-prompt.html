1.1. note¶

  1. We use these for all the examples below:

    • folder path (from) = E:\folder1
    • folder path (to) = E:\folder2
    • original file path = E:\folder1\file1.txt
    • EXE file = E:\PeaZipPortable\PeaZipPortable.exe

  2. Command that we will use:

    CLIP
ECHO
PAUSE
SET
FOR
FORFILES
PING
IF EXIST
DIR
TYPE
RMDIR
DEL
MKDIR
RENAME
MOVE
XCOPY
COPY
TASKKILL
START

1.2. Quick way to open Command Prompt in current folder¶

  1. Just type cmd at the address bar or
  2. Shift + right-click then click Open command window here or
  3. For Windows 10, click File > Open command prompt

1.3. Copy to clipboard¶

List all the files and folders inside current folder and then copy to clipboard

DIR /B | CLIP

1.4. Save to a file¶

List all the files and folders inside current folder and then save to list.txt

DIR /B > list.txt

1.5. Display text¶

  1. Turns command-echoing off

    @ECHO OFF

  2. display hello world

    @ECHO OFF
ECHO hello world
PAUSE

1.6. Read user input and branching¶

ECHO Select a file:
ECHO     1 = one.xml
ECHO     2 = two.xml
ECHO ?
SET /P file=
IF %file%==1 (
ECHO you select one.xml
)
IF %file%==2 (
ECHO you select two.xml
)

1.7. FOR loop¶

  1. Prints 10 then 11 then 12

    FOR %%A IN (10 11 12) DO ECHO %%A

1.8. Do operation to all selected files¶

FORFILES /M *.txt /C "CMD /C ECHO @FNAME is a TXT file"

1.9. About currently running batch file¶

1.9.1. get parent folder¶

%~dp0

1.9.2. get filename without extension¶

%~n0

1.9.3. get filename with extension (base)¶

%~nx0

previously, I used:

%~n0%~x0

1.9.4. other variables¶

just study the notes of:

FOR /?

1.10. Suspends processing¶

1.10.1. pause¶

PAUSE

1.10.2. Pause for 5 seconds¶

PING 1.1.1.1 -n 1 -w 5000 > NUL

1.11. List all the files and folders inside a folder¶

1.11.1. a given folder¶

DIR "E:\folder1" /B

1.11.2. current folder¶

DIR /B

1.11.3. current folder and then copy to clipboard¶

DIR /B | CLIP

1.11.4. current folder and then save to list.txt¶

DIR /B > list.txt

1.11.5. also include items in subfolders¶

DIR "E:\folder1" /B /S

1.11.6. list only files¶

DIR "E:\folder1" /A:-D /B

1.11.7. list only folders¶

DIR "E:\folder1" /A:D /B

1.11.8. sort files by extension¶

DIR /A:-D /B /O:E

1.12. Check if exist¶

1.12.1. folder¶

IF EXIST "E:\folder1" (
ECHO folder exist
)

1.12.2. file¶

IF EXIST "E:\folder1\file1.txt" (
ECHO file exist
)

1.13. Show the content of a text file¶

TYPE "E:\folder1\file1.txt"

1.14. Delete¶

1.14.1. folder¶

IF EXIST "E:\folder1" (
RMDIR /S /Q "E:\folder1"
)

1.14.2. file¶

IF EXIST "E:\folder1\file1.txt" (
DEL /F /Q "E:\folder1\file1.txt"
)

1.15. Create¶

1.15.1. folder¶

MKDIR "E:\folder1"

1.15.2. empty file¶

TYPE NUL > "E:\folder1\file1.txt"

1.16. Rename¶

1.16.1. folder¶

RENAME "E:\folder1" "folder2"

1.16.2. file¶

RENAME "E:\folder1\file1.txt" "file2.txt"

Notice that when renaming folder/file, there is no path info in the destination

1.17. Concatenate all CSS files into generated.css¶

IF EXIST generated.css (
DEL /F /Q generated.css
)
IF EXIST generated.css.temp (
DEL /F /Q generated.css.temp
)
TYPE *.css > generated.css.temp
RENAME generated.css.temp generated.css

1.18. Move/Cut¶

1.18.1. folder¶

MOVE /Y "E:\folder1" "E:\folder2"

folder1 is now moved to folder2. folder1 is now inside folder2.

1.18.2. file¶

MOVE /Y "E:\folder1\file1.txt" "E:\folder2\file1.txt"

1.19. Copy¶

1.19.1. folder¶

  1. Displays full source and destination file names while copying (/F):

    XCOPY "E:\folder1" "E:\folder2\folder1" /E /C /I /F /G /H /R /K /Y

  2. Displays files that would be copied (/L):

    XCOPY "E:\folder1" "E:\folder2\folder1" /E /C /I /F /L /G /H /R /K /Y

  3. Does not display file names while copying (/Q):

    XCOPY "E:\folder1" "E:\folder2\folder1" /E /C /I /Q /G /H /R /K /Y

  4. Creates directory structure, but does not copy files. (/T):

    XCOPY "E:\folder1" "E:\folder2\folder1" /E /C /I /F /G /H /R /T /K /Y

1.19.2. file¶

COPY /V /Y "E:\folder1\file1.txt" "E:\folder2\file1.txt"

1.20. End/Kill task¶

1.20.1. End task all opened Command Prompt¶

TASKKILL /IM cmd.exe /T /F

1.20.2. End task Microsoft Excel¶

TASKKILL /IM EXCEL.EXE /T /F

1.20.3. End task explorer.exe and restart it¶

@ECHO OFF
TASKKILL /IM explorer.exe /F
START explorer.exe

Do not use /T for explorer.exe.

START explorer.exe is needed to close Command Prompt; explorer.exe is just not enough.

1.21. Starting an EXE file¶

1.21.1. Normal EXE file with GUI¶

START "" "E:\PeaZipPortable\PeaZipPortable.exe"
