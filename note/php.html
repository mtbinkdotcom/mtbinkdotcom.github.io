7.1. Common script¶

  • loop every file inside a folder

    if ( file_exists($folder_path) ) {
    if ( $dh = opendir($folder_path) ) {
        while ( ( $f = readdir($dh) ) !== false ) {
            if ( $f != "." && $f != ".." ) {
                // $f is now a filename
            }
        }
        closedir($dh);
    }
}

  • for loop

    for ($i = 0, $ii = count($arrayName); $i < $ii; $i++){
    $ti = $arrayName[$i];
}

for ($j = 0, $jj = count($arrayName); $j < $jj; $j++){
    $tj = $arrayName[$j];
}

for ($k = 0, $kk = count($arrayName); $k < $kk; $k++){
    $tk = $arrayName[$k];
}

  • string replacement

    $a = "mtbink.com";
$a = str_replace(".", "dot", $a); // mtbinkdotcom

  • get the timestamp of a file

      date("d F Y (l) h:i:s A", filemtime($file_path) );
  // 19 July 2014 (Saturday) 05:14:08 AM
  // sometimes: %d %B %Y (%A) %H:%M:%S %p %Z

gmdate("c", filemtime($file_path)); // 2014-07-18T21:14:08+00:00

  • string length

    strlen("mtbink"); // 6

  • split/join string to array

    explode("=", $string); // split
implode("=", $string); // join

  • array count

    count($array);

  • search string position:

    strpos("mtbink","b"); // returns 2

  • search item position in an array

    in_array($value, $arrayName);

  • substring

    substr("abcdef", -2); //  returns "ef"
